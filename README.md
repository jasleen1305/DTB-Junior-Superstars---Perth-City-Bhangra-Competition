# DTB-Junior-Superstars---Perth-City-Bhangra-Competition

This repository hosts a minimal static website for the DTB Junior Superstars — Perth City Bhangra Competition.

Live site (after first push to `main` and GitHub Actions run):

- https://jasleen1305.github.io/DTB-Junior-Superstars---Perth-City-Bhangra-Competition/

To preview locally, open the file `public/index.html` in a browser or run a simple static server such as:

```bash
python3 -m http.server --directory public 8000
```
